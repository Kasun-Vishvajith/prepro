# prepro

A Python preprocessing library for data science pipelines.

## Installation

You can install `prepro` via pip:

```bash
pip install prepro
```

## Quick Start

```python
import prepro

# Basic usage example
print(prepro.__version__)
```

## Features

- Standardized data preprocessing functions.
- Clean, modular API designed for data science workflows.
